# AWS Next.js Portfolio Project

This is a **full-stack cloud application** built with **Next.js and React**, deployed on **AWS Amplify**. 

## Features
- Server-side rendering with Next.js
- API routes (AWS Lambda)
- Authentication (planned with Amazon Cognito)
- Database integration (planned with DynamoDB)
- Fully responsive UI with Tailwind CSS

## Tech Stack
- Next.js + React
- AWS Amplify
- AWS Lambda
- DynamoDB
- Tailwind CSS
